# [Conjure Dex](https://conjure-cards.app)
Public-facing breakdown of Conjure Dex, a Flutter progressive web app and digital encyclopaedia for the Conjure card game.

## Key Features
-  A digital recreation of every physical game card in the Conjure game
-  Every physical game card has a QR code linking directly to the card's digital page - <b>whether the app is installed or not</b>
-  All icons have detailed 'press for explanation' popups, providing on-demand explanations of any symbol in the game
   
   <img width="3508" height="2250" alt="Scan, app, popups" src="https://github.com/user-attachments/assets/192f7813-bb4f-43a6-8f03-4cf3ee2492b3" />

-  Scanning QR codes can be performed either through the app's internal scanner <b>or</b> any default phone camera
-  App includes 'hidden' cards which can be unlocked by scanning their QR code, to incentivise card trading and collection
  
-  Recently viewed cards are tracked through local storage for quick navigation
-  Guide section explains the game's icons and rules
